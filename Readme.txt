The files contains the sample data of 48 typical days. ： the historical forecasts/true profiles of the PV/wind power generation and load profiles in 5-minute time resolution (288 sampling points) for one day.

File description:
‘d_DA_5min’:  historical forecasts of load 
‘w_DA_5min’: historical forecasts of wind power generation 
‘s_DA_5min’: historical forecasts of PV power generation 
‘d_RT_5min’: historical true profiles of load
‘w_RT_5min’: historical true profiles of wind power generation
‘s_RT_5min’: historical true profiles of PV power generation
